Building RESTful Services with Node.js
What is Node.js?
  • Open Source
  • Server-side JavaScript runtime
  • Built on Google V8 (chrome's JavaScript engine)
  • Single-threaded event loop (just like a browser)
  • All I/O is asynchronous (non-blocking)
You already do asynchronous JavaScript!
$.get('/somewhere', function(data)) { 
    alert(data);
});

...

setTimeout('doSomething()', 2000);
Strong Community
  • Central registry for community contributed modules (called NPM, npmjs.org)
  • 7,000+ modules already contributed
  • Weekly podcast to communicate latest developments (called NodeUp, nodeup.com)
  • Annual conferences and regional meetup groups (ex. Node.Philly, node.ph)
What are RESTful Services?
  • User HTTP methods explicitly
    • POST: create a resource
    • GET: retrieve a resource
    • PUT/PATCH: update a resource
    • DELETE: delete a resource
  • Be stateless
  • Use XML or JSON protocol (preferrably JSON)
  • Use descriptive URIs
